# Social-Media-Language-Analysis-for-Anxiety-Subtype-Classification
Social Media Language Analysis for Anxiety Subtype Classification is an NLP-based application developed to classify Reddit posts into different anxiety subtypes using machine learning techniques. The tools involved are Python, NLP, BERT,Streamlit and Machine Learning.
